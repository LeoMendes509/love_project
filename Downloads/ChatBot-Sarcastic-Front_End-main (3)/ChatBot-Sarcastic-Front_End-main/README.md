# ChatBot Sarcastic Front-End

## Descrição / Description
**PT:** Front-End de um chatbot sarcástico em React e TypeScript. Permite login de usuário, chat dinâmico e interface responsiva. Foco em integração com back-end e Context API.  
**EN:** Front-End of a sarcastic chatbot using React and TypeScript. Supports user login, dynamic chat, and responsive interface. Focused on back-end integration and Context API.

## Tecnologias / Technologies
**PT:** React, TypeScript, Vite, Context API, CSS moderno  
**EN:** React, TypeScript, Vite, Context API, modern CSS

## Uso / Usage
**PT:** Abra a aplicação no navegador, faça login e interaja com o chatbot.  
**EN:** Open the app in your browser, log in, and interact with the chatbot.
